== Description ==

Not Sure plugin allows your customers to ask questions and or answer predefined questions, in example a shop website could use the plugin to place not sure buttons in the product pages.

Not Sure features:

*   Easy to install and use
*   Create as many not sure buttons as needed
*   Configure each button per your needs
*   Unlimited number of questions
*   Customer can send you messages
*   Email gets sent to administrator and other emails if set
*   Opens in a pop-up
*   Shortcode ready

Help your customers get in touch with you easily

== Installation ==

Installing Not Sure plugin

1. Install the plugin as any other Wordpress plugin
2. Create Not Sure buttons per your needs
3. Place shortcodes of buttons in your posts etc
